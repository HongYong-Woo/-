1. 메서드(method) : 반복되는 특정 작업을 수행하기 위한 코드의 집합
*c언어에서는 함수(function)라고 함

필요성 :  특정 작업을 빠르게 작업 가능

자바에서는 메서드를 두가지로 분류
-표준 라이브러리 (J2SE API)에서 제공하는 메서드
-사용자 정의 메서드

1-1. 메서드를 사용함으로써 우리가 얻는 이점
	-소스를 재사용성(reusing) : 중복 코드 방지
	-소스의 가독성 : 한눈에 들어오게 코드를 관리 할 수 있음.
	-소스의 유지보수성 : 리팩토링을 쉽게 함.

2. 메서드의 기본 구조
	접근 제한자(Access Modifier) 선택사항(public , protected, default, private)

	*반환 유형(return type) : 메서드의 리턴 유형을 결정, void, int,double, 등....
	 void(return 없음) 그외( 정한 return 값이 반드시 필요)

	*매서드명(method name): 메서드가 호출될때 사용되는 이름
	 이름을 지을때 : 카멜표기법(소문자로 시작하여 다른 단어 나올떄 대문자 ex) dicePlay)
	  -google style-quide (소문자도 시작하여 명명)
	  -메서드의 명명은 반드시 동사 형태
	  -추가하다 기능의 이름을 지어보기public int add(int a, int b)
	  -이름을 조회하기		public String getName()
	  -상세주소 조회하기	public String getAddress()
	*만약에 메서드명에 3개이상 단어가 포함 : 동사로 시작하고 그뒤에 형용사 또는 명사가 오는 경우

	*매개변수 목록(parameter )
3. 메소드의 사용 방법
	메서드 오버로딩
	*메서드 명이 같지만 매가변수나 자료형의 개수가 다름 (다형성의 기법 중 하나)
