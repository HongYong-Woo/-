 *객체지향 프로그래밍 -> 클래스 단위로 프로그래밍 하는것

-추상화 : 일반화, 단순화 시키는것

main class -> 시작 class (메모리 점유)

new 클래스명  -> 힙메모리에 할당

프로그래밍 3대 요소 : 변수(Variable), 자료형(Datatype), 할당(Assign)
변수: 데이터를 저장하는 메모리 공간의 이름
자료형: 변수의 크기와 종류를 결정하는 것
할당 : 변수를 저장하는 것

변수의 종류	저장공간
1. 인스턴스 변수	힙
2. 스태틱 변수	메소드
3. 지역변수		스택
4. 매개변수		스택

1byte = 8bit
