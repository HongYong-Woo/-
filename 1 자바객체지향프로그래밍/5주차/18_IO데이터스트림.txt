데이터 입출력
-자바에서 데이터를 입력하고 출력기능을 제공하는 라이브러리(API)를 활용
1) 파일 입/출력
2) TCP / UDP 기반의 네트워크 환경에서 데이터 입/출력

========
1.입력 스트림이란? : 데이터가 들어오면 입력 스트림
2.출력 스트림이란> : 데이터가 나가면 출력 스트림
3,입력/출력 스트림이 데이터를 처리하는 단위
	-Byte
	 입력 스트림 최상위 클래스 : InputStream	(추상)
	 출력 스트림 최상위 클래스 : OutputStream	(추상)
	-문자
	 입력 스트림 최상위 클래스 : Reader	(추상)
	 출력 스트림 최상위 클래스 : Writer	(추상)
